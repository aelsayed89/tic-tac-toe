# Tic-Tac-Toe Game

A modern, interactive Tic-Tac-Toe game built with Next.js 16, React 19, TypeScript, and Tailwind CSS.

## Features

- Clean, responsive UI
- Two-player gameplay
- Real-time game state updates
- Winner detection
- Reset functionality

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to play the game.

## Building for Production

```bash
npm run build
npm start
```

## Deploy on Vercel

The easiest way to deploy this app is using [Vercel](https://vercel.com):

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/aelsayed89/tic-tac-toe)

1. Push your code to GitHub (already done!)
2. Go to [Vercel](https://vercel.com)
3. Import your GitHub repository `aelsayed89/tic-tac-toe`
4. Vercel will automatically detect Next.js and configure the build settings
5. Click "Deploy"

Your app will be live in minutes!
